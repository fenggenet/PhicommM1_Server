# 说明

Phicomm悟空M1官方服务器关闭,WIFI图标一直闪烁,最开始只是想写个监听9000端口的服务器,让它的WIFI图标不闪,后来写着写着就加上了把获取的数据保存到MYSQL,再后来,即然都保存到MYSQL了,干脆再写个前端吧.Python小白,代码比较乱.
# 安装
首先安装必要的环境

`pip install Flask`

`pip install sqlalchemy`

`pip install pymysql`

`pip install mysql-connector`
# 运行

`chmod a+x ./run.sh`
